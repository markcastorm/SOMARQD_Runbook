  Based on a detailed review of information.txt, the two reference scrapers from the other runbooks (scraper.py and scraper.py), and the tests we just ran, here is the analysis of the Selenium fallback  
  method and what is needed to make it work.

  ### Will the Selenium Method Work? (Why We Had Issues)                                                                                                                                                          
                                                                                                                                                                                                                  
  Currently, if the direct downloads fail, the Selenium fallback method as originally written will also fail due to three main gaps:                                                                              
                                                                                                                                                                                                                  
  1. ChromeDriver Version Mismatch (Resolved in our test):                                                                                                                                                        
      • Problem:  undetected_chromedriver  automatically fetches the latest driver (version 150), but the user's local Chrome browser is version 149. This results in a session creation crash.                   
      • Solution: We must dynamically query the Chrome major version from the Windows registry ( winreg ) or Linux shell (exactly like scraper.py does) and pass it to  version_main  in  uc.Chrome .         
  2. Browser Cookie Transfer (Missing in current code):                                                                                                                                                           
      • Problem: If direct requests are blocked by the site, the Selenium browser bypasses the block, but then the script downloads the PDF using a fresh  requests  session with no cookies.                     
      • Solution: We must capture the cookies from the active Selenium driver and inject them into  requests  (similar to  session_cookies = {c['name']: c['value'] for c in driver.get_cookies()}  used in       
      AUSFEDGOVCF ).                                                                                                                                                                                              
  3. Fragile Link Selectors:                                                                                                                                                                                      
      • Problem: The current Selenium script targets links using a hardcoded URL fragment (e.g.  sb0485 ). If the Treasury changes their URL naming pattern, this fails.                                          
      • Solution: We must dynamically search the archive page's HTML table using the table's hierarchical relationships (matching the year's  id  header with the corresponding quarter cell's  headers  attribute
      and text like  "2nd Quarter" ), scroll it into view, and click it via JavaScript.                                                                                                                           
                                                                                                                                                                                                                  
  ──────                                                                                                                                                                                                          
  ### Key Scraper Architecture Features from Reference Runbooks                                                                                                                                                   
                                                                                                                                                                                                                  
  To align the scraper with the other production runbooks and fully satisfy the  information.txt  requirements, we should incorporate these design choices:                                                       
                                                                                                                                                                                                                  
  #### 1. Adaptive, Future-Proof Navigation (Inspired by  AUSFEDGOVCF )                                                                                                                                           
                                                                                                                                                                                                                  
  • XPath & Table Headers: Instead of matching URL substrings, search for the quarter link under the specific year table cell.                                                                                    
    <!-- HTML Structure -->                                                                                                                                                                                       
    <tr><th id="2026">2026</th></tr>                                                                                                                                                                              
    <tr><th headers="2026"><a href="...">2nd Quarter</a></th></tr>                                                                                                                                                
    We can write an XPath query like  //th[@headers='{year}']/a[contains(text(), '{quarter_label}')]  to find the exact cell dynamically.                                                                         
  • Scroll & JS Click: Scroll the element into the center of the viewport and trigger the click using  driver.execute_script("arguments[0].click();", element) .                                                  
                                                                                                                                                                                                                  
  #### 2. Legacy Direct PDF Redirection (e.g., 2015 Q3)                                                                                                                                                           
                                                                                                                                                                                                                  
  • Embedded Link Extraction: As described in  information.txt , some years link directly to a "Financing Estimates" PDF wrapper rather than a press-release page.                                                
  • We can use  PyPDF2  or  pdfplumber  to check if the downloaded PDF is a wrapper page. If so, parse its  /Annots  (URI annotations) to extract the embedded link to the actual "Sources and Uses Table" PDF,   
  and download that instead.                                                                                                                                                                                      
                                                                                                                                                                                                                  
  #### 3. Multi-Quarter Download Loop
  
  • List-Based Output: Currently, the orchestrator only processes a single PDF dict. If  config.TARGET_YEAR  is specified and  config.TARGET_QUARTER  is  None , the scraper should identify and download all     
  active quarters for that year.
  • The downloads will be placed in nested directories:  downloads/<timestamp>/<year>/Q<quarter>/ .
  • The scraper's  download()  entry point will return a list of dictionaries (just like  AUSFEDGOVCF  does), allowing the orchestrator to loop through and extract data for each downloaded quarter.
